原文 Headroom — The context compression layer for AI agents.

60–95% fewer tokens across library, proxy, MCP, and agent-wrap modes. 6 compression algorithms. Local-first. Reversible compression.
中文 Headroom —— AI agent 的上下文压缩层。

在 library、proxy、MCP 和 agent-wrap 四种模式下减少 60-95% 的 token。6 种压缩算法。本地优先。可逆压缩。
原文 Five main modes:
1. Library — inline compress(messages) in Python or TypeScript
2. Proxy — drop-in HTTP proxy on port 8787 requiring zero code changes
3. Agent wrap — wraps Claude, Codex, Cursor, Aider, Copilot, OpenCode via a single command
4. MCP server — exposes headroom_compress, headroom_retrieve, headroom_stats tools
5. Cross-agent memory — shared store across Claude, Codex, Gemini with auto-dedup
中文 五种主要模式：
1. Library（代码库模式）—— 在 Python 或 TypeScript 中直接调用 compress(messages)
2. Proxy（代理模式）—— 在 8787 端口启动一个 HTTP 代理，一行代码不用改
3. Agent wrap（Agent 包装模式）—— 一条命令同时包装 Claude、Codex、Cursor、Aider、Copilot、OpenCode
4. MCP server（MCP 服务器模式）—— 暴露 headroom_compress、headroom_retrieve、headroom_stats 工具
5. Cross-agent memory（跨 Agent 记忆）—— Claude、Codex、Gemini 之间共享存储，自动去重
原文 Architecture: ContentRouter detects content type and routes to SmartCrusher (JSON), CodeCompressor (AST-aware compression), or Kompress-base (HuggingFace text model). CacheAligner stabilizes prefixes for KV cache hits. CCR stores originals locally for on-demand retrieval.

headroom learn: Mines failed sessions and writes corrections — output token reduction trims model verbosity.
中文 架构：ContentRouter（内容路由器）检测内容类型，然后路由到 SmartCrusher（处理 JSON）、CodeCompressor（AST 感知的代码压缩）或 Kompress-base（基于 HuggingFace 文本模型）。CacheAligner 稳定前缀以提高 KV cache 命中率。CCR 在本地存储原始内容供按需检索。

headroom learn：挖掘失败的 session 并自动写修正规则——还能压缩模型输出，trim 掉冗余的"废话"。
原文 Benchmark results:
• Code search: 92% savings (17,765 → 1,408 tokens)
• SRE incident debugging: 92% savings (65,694 → 5,118)
• Accuracy preserved: GSM8K (±0.000), TruthfulQA (+0.030), SQuAD v2 (97%), BFCL (97%)
中文 基准测试结果：
• 代码搜索：节省 92% token（17,765 → 1,408）
• SRE 事故调试：节省 92% token（65,694 → 5,118）
• 准确度保持：GSM8K（±0.000，无变化）、TruthfulQA（+0.030，反而提升）、SQuAD v2（97%）、BFCL（97%）
原文 Comparison: RTK, lean-ctx, Compresr, Token Co., OpenAI Compaction — Headroom is the only local, reversible option spanning all content types (JSON, code, text, logs, RAG chunks).

Agent compatibility: Claude Code, Codex, Cursor, Aider, Copilot CLI, OpenClaw, OpenCode, Cortex Code.
中文 对比：RTK、lean-ctx、Compresr、Token Co.、OpenAI Compaction —— Headroom 是唯一一个本地运行、可逆压缩、覆盖所有内容类型（JSON、代码、文本、日志、RAG chunks）的方案。

Agent 兼容性：支持 Claude Code、Codex、Cursor、Aider、Copilot CLI、OpenClaw、OpenCode、Cortex Code。
