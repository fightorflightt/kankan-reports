同类资料速览

想搞懂 Claude Code / Agent 架构，除了它还有哪些选择

官方文档 Anthropic 官方的 Claude Code 文档。最权威、最新，但只讲"怎么用"，不讲"为什么这么设计"。适合上手，不适合理解内部。
李博杰博客 《一场泄露看懂 Claude Code：Harness 是让 Agent 干活靠谱的关键》。一篇深度长文，是这本书的"轻量版"——想快速了解核心观点、不想读 42 万字的人，先看这篇。（01.me）
各类逆向分析文 源码泄露后社区涌现了一批"Claude Code 源码逆向"文章/指南，零散、深浅不一。胜在新鲜及时，输在不成体系——这本书相当于把这些零散讨论系统化、结构化了。
框架官方教程 LangChain / AutoGen / CrewAI 等自家文档。教你用具体框架搭 Agent，能直接动手，但绑定单一框架、且不会带你看"标杆产品"的设计取舍。
📌
这本书适合：

想系统、深入理解一个生产级 Agent 到底怎么搭的人——架构师、要带队自研 Agent 的工程负责人、做研究的人。也适合想用一套统一框架建立"Agent 全局观"的技术读者。


不适合：

只想快速上手 Claude Code 的使用者（看官方文档更快）；想找"复制粘贴就能跑"的代码模板的人（这是讲思想的书，不是脚手架）；以及完全无技术背景、只想要结论的人（建议直接看本报告 + 李博杰那篇长文就够了）。
