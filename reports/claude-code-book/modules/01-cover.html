这是什么？

这不是一个软件，而是一本 42 万字的免费在线技术书。它干的事只有一件：把 Claude Code（Anthropic 官方的 AI 编程助手）这个"AI 干活机器"彻底拆开，告诉你它内部是怎么搭起来的——AI 怎么一轮轮自己跟自己对话把活干完、怎么管权限不乱来、怎么在"记忆有限"的情况下还能记住该记的事。

书名"御舆"取自古代造马车的典故：马车里有承载乘客的车厢（舆）、定方向的辕、传力的辐……每个零件各司其职合起来车才能跑。作者用这个比喻说：搭一个 AI Agent 也一样，那个把所有零件托起来、让 AI 真正转起来的"运行时骨架"，就叫 Agent Harness（可以理解成 AI 助手的"底盘 + 神经系统"）。市面上都在教你怎么"用"AI，这本书带你拆开它看里面。

应用场景

想自己做一个 AI Agent 产品的人：你不想只会调 API，想知道一个"生产级"的 AI 助手底层骨架长什么样、有哪些坑，把它当设计参考蓝图。
重度用 Claude Code / Cursor 这类工具的人：搞懂它为什么这么设计，从而把工具用得更狠——知道它什么时候会"压缩记忆"、权限为什么要拦你。
技术管理者 / 架构师：想给团队选型或自研 Agent 框架时，手里有一张"设计决策地图"，知道每个选择的取舍在哪。
研究者 / 想写文章的人：拿它当一份系统性的二手资料，书里有 139 张架构图可以直接引用。
