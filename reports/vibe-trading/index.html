这是什么？

Vibe-Trading 是香港大学数据智能实验室（HKUDS）开源的"个人交易研究工作台"——你用大白话提问，AI 帮你把整条"做交易研究"的流水线跑完：找行情数据 → 生成策略 → 回测验证 → 出报告。一句话提问、一条命令就能动起来。

说人话就是：以前散户想验证一个交易想法（比如"某只股票用某种买卖规则,过去两年能赚多少"），得自己懂编程、懂量化、搭一堆数据和回测系统。Vibe-Trading 把这套"机构级"的活儿,变成了"你说想法、AI 干活"。它覆盖 A股/港股/美股/加密/期货/外汇,还能复盘你自己真实的交易记录。注意:它定位是研究、模拟、回测工具,不是荐股、也不替你保管钱。

应用场景

散户验证交易想法：脑子里有个买卖规则，想知道"过去这么做到底赚不赚"，用一句话让 AI 跑回测、出夏普比率和最大回撤，不用自己写代码。
复盘自己的真实交易：把券商导出的交易流水（同花顺/东财/富途）传进去，AI 分析你的交易习惯和坏毛病（追涨、过早止盈），还告诉你"严格按你自己的规则本来能多赚多少"。
量化学习 / 教学：自带 456 个现成的量化因子和 79 个金融技能，新手能直接拿来学、跑、对比，是低门槛入门量化的好教材。
多智能体投研团队：让一组 AI 扮演"投资委员会/量化台/风控会"互相辩论，给一只股票做多空对辩再出结论——一个人也能开"投研团队会"。

解决了什么痛点

为什么港大实验室要做这个？它替散户解了什么气？

想认真做交易研究的普通人，通常卡在这三关

第一关：想验证一个交易想法，门槛高得吓人。普通人脑子里常有念头——"如果某只股票跌破某条线就买、涨上去就卖,这么操作到底行不行?"可要真验证它,你得会写代码、懂量化、还要自己接行情数据、搭回测系统。这一整套"基础设施"把绝大多数有想法的人挡在了门外。Vibe-Trading 的核心动机就是:让你用大白话说出想法,AI 把后面又脏又累的活全包了。

第二关：数据源又贵又乱、还经常被封。做量化最头疼的之一是搞数据——A股、港股、美股、加密各有各的接口,免费的容易被限流封 IP,好用的要付费。Vibe-Trading 接了 18 个数据源,还做了"智能兜底":一个源挂了自动换下一个,优先用永远不封的免费源,让你"零配置、不用管"就能拿到多市场行情。

第三关：自己交易亏在哪，说不清也改不掉。很多人炒股年年亏,却讲不清到底是规则不行、还是自己手贱（追涨杀跌、过早止盈）。它的"影子账户"功能直接戳这个痛点:把你的真实交易流水导进去,AI 帮你提炼出你"本来想执行的规则",再用这套规则跑一遍回测——对比之下,你一眼能看到"如果我老老实实按规则做,本来能多赚多少钱",哪些钱是被自己的坏习惯亏掉的。

## 💡 What Is Vibe-Trading?
Vibe-Trading is an open-source research workspace for turning finance questions into runnable analysis. It connects natural-language prompts to market-data loaders, strategy generation, backtest engines, reports, exports, and persistent research memory.
It is designed for research, simulation, and backtesting — and, when you choose, autonomous trading through a broker you authorize yourself (e.g. Robinhood Agentic Trading). It holds no funds and never trades outside the limits you set, and you can halt it instantly.

＃＃ 💡 什么是 Vibe-Trading？

Vibe-Trading 是一个开源的研究工作台，把金融问题变成可运行的分析。它把自然语言提示词连接到市场数据加载器、策略生成、回测引擎、报告、导出，以及持久的研究记忆。

## 👥 Shadow Account
Shadow Account starts from your own trading records instead of a generic strategy template.
Upload a broker export, let the agent summarize your behavior, then compare the actual trading path with a rule-based shadow strategy.
1. Read your journal → Parses broker exports from 同花顺, 东方财富, 富途, generic CSV.
2. Profile your behavior → Holding days, win rate, PnL ratio, drawdown, disposition effect, overtrading, momentum chasing, anchoring.
3. Extract your rules → Turns recurring entries/exits into an explicit strategy profile.
4. Run the shadow → Backtests the rules, highlights rule breaks, early exits, missed signals, alternative trade paths.
5. Deliver the report → HTML/PDF report to inspect, archive, or refine later.
中文

＃＃ 👥 影子账户（Shadow Account）

影子账户从你自己的交易记录出发，而不是一个通用的策略模板。

上传一份券商导出文件，让 agent 总结你的行为，然后把你真实的交易路径和一个基于规则的影子策略做对比。

1. 读取你的流水 → 解析来自同花顺、东方财富、富途以及通用 CSV 格式的券商导出。

2. 给你的行为画像 → 持仓天数、胜率、盈亏比、回撤，以及处置效应、过度交易、追涨、锚定等检查。

3. 提取你的规则 → 把反复出现的买入/卖出动作，变成一份明确的策略画像，而不是含糊的总结。

4. 运行影子策略 → 回测这套提取出的规则，并高亮出违反规则之处、过早离场、错过的信号和其他可能的交易路径。

5. 交付报告 → 产出一份可查看、可存档、可在后续会话里继续打磨的 HTML/PDF 报告。

04

创新点

市面上量化工具一堆，它凭什么 30 天涨到几千 star？

和常规做法不一样的地方

过去的量化工具（如 Qlib、Backtrader）是给"会写代码的量化工程师"用的——你得自己编程、配数据、调框架。Vibe-Trading 的根本不同是"把整条研究流水线交给 AI agent，用户只管说话"。它不是又一个回测库,而是一个"会用一堆专业工具的 AI 研究员":你提需求,它自己选技能、拉数据、写策略、跑回测、出报告。这把量化的门槛从"程序员"直接降到了"会提问的人"。

🗣️
"一句话 → 完整研究"的范式

最核心的创新：自然语言进、可交付的报告/策略/回测出。它把"数据→策略→回测→报告"压进一条命令。别的工具给你"零件",它直接给你"装好的成品流程"。

👥
影子账户：拿你自己的交易当镜子

这是很少见的角度。不教你一个"通用好策略",而是从你真实流水里提炼出"你本来想做的规则",再回测对比——精准告诉你"哪些钱是被你自己的坏习惯亏掉的"。把抽象的"反思"变成了可量化的对照。

🐝
多智能体投研团队（Swarm）

29 个预置"团队":投资委员会做多空辩论、量化台从选股到回测、风控会签字。一个人也能开一场"机构式投研会议",让不同角色的 AI 互相质疑后再出结论,比单个 AI 更可靠。

📡
18 源智能兜底，零配置抓全球行情

把"搞数据"这个量化最大的脏活做成了开箱即用:18 个数据源按"封禁风险"排队,一个挂了自动换下一个,免费源优先。覆盖 A股/港股/美股/加密/期货/外汇,不用 key 就能跑。

🧬
456 个现成因子 + 一行命令跑基准

内置了学术界和业界经典的 456 个 alpha 因子(微软 Qlib、Kakushadze 101、国君 191 等),一行命令就能在你的股票池上批量评测、自动标出"还有效/已反转/已失效"。等于把量化研究的"弹药库"也配齐了。

🔒
把"实盘"关进笼子里的克制设计

它能接券商实盘,但处处设防:默认只读、要你亲自授权、只在你设的额度内交易、有一键急停、不持有你的钱。这种"能力强但主动给自己上锁"的设计,在交易类项目里是难得的负责任态度。

灵感卡片

看完这个项目我想到的产品点子——不一定用它的代码

灵感 1
"拿你自己的数据当镜子"是个万能产品母题

影子账户最妙的不是量化,是"从你的真实记录里提炼出你本来的意图,再对比理想执行,告诉你差在哪"。这个套路能套到任何"用户想变好却说不清问题在哪"的场景——健身、学习、时间管理、内容创作。把模糊的"反思"变成可量化的对照,是很强的产品钩子。

灵感 2
"一句话 → 成品流程"是 AI 产品的高级形态

它不给零件、直接给装好的流程:自然语言进,完整报告出。这对你做独立小工具特别有启发——凡是"多步骤、用户嫌麻烦"的事,别给一堆按钮让用户自己拼,试着让用户一句话说需求、AI 把整条流水线跑完直接交付成品。这是从"工具"升级到"代办"的关键一跳。

灵感 3
"多角色 AI 互相辩论"提升结论质量

它让投资委员会多空对辩再出结论。任何"AI 给建议"的功能,与其让一个 AI 直接拍板,不如让"正方/反方/挑刺者"几个角色先互怼一轮——结论更稳、更经得起推敲,用户也更敢信。这是个可以直接复用的产品设计模式。

灵感 4
"智能兜底链"=用户零感知的稳定性

18 个数据源按风险排队、一个挂了自动换下一个、用户完全无感。任何依赖外部接口的产品都该学这招:别让上游抽风直接糊用户一脸报错,准备好备用源、优雅降级。稳定到"用户从不知道你救过场",才是真功夫。

灵感 5
用更新日志把"我们一直在进步"讲给用户听

它 README 顶部是一长串每日更新日志,30 天天天发版,营造出极强的"团队很拼、产品在飞速进化"的信任感。任何产品的更新页/动态,都可以把"我们改了什么"持续、显眼地晒出来——让用户真切感到产品在被认真打磨,这本身就是留存。

灵感 6
"能力强但主动上锁"反而赢得信任

它能接实盘,却处处设防:默认只读、要你亲自授权、一键急停、不碰你的钱。在涉及钱或敏感操作的功能上,把"安全护栏"做得显眼,不但不减分,反而是最强的信任牌。克制,有时比强大更打动人。

产品化可行性

一个不写代码的 PM，能不能用这个项目做出东西来？

低门槛
上手难度

它本身就是"用大白话提问"的产品。装好后你就用中文/英文说需求,它干活。对会用命令行的人几乎零代码;真正的门槛在"装"那一步,不在"用"那一步。

中等
依赖链

需要 Python 3.11+ 或 Docker,再加一个大模型的 API key(或本地 Ollama)。行情数据有免费源、零配置;想接券商实盘则要额外授权配置。整体免费开源。

低门槛
Vibe Coding 适配度

天生就是为"跟 AI 对话用工具"设计的,还自带 MCP 接口能挂到 Claude Desktop。让 Claude Code 帮你装、帮你跑,是最顺的打开方式。

如果你要用，大概需要什么

最省事试一把：用 Docker 路径,一条命令起服务,浏览器打开就有 Web 界面,2 分钟能跑起来——这是对非技术人最友好的方式。然后准备一个大模型 key(它推荐 DeepSeek 等性价比模型,成本很低)。

真正用起来：时间上当天就能体验;钱主要是大模型的调用费(用性价比模型很便宜,行情数据免费);前置知识需要一点点交易常识(什么是回测、夏普、回撤),但完全不用懂编程。让 AI 帮你装环境能省掉绝大部分麻烦。

⚠️
注意：它是"研究工具"，不是"印钞机"

作者在 README 顶部和免责声明里反复强调:它只用于研究、模拟、回测,不是投资建议。历史回测跑得好看 ≠ 真金白银能赚——这是量化最大的坑(过拟合、幸存者偏差)。它虽能接券商实盘,但作者明说"实验性、没用真账户验证过、风险自负"。把它当学习和验证想法的工具,别把回测曲线当未来收益保证。另外:模型一定要选能可靠调用工具的(别用小模型),否则它会"凭记忆瞎编"而不是真去跑数据。

07

同类项目速览

它在"量化 / 交易工具"生态里站什么位置

Qlib（微软） 最有名的开源 AI 量化平台，功能强、偏机构和研究员，要会写 Python。Vibe-Trading 反而把 Qlib 的 158 因子收进了自己的因子库——它定位是"上层的 AI 操盘手"，让你不写代码也能用上这类能力。
Backtrader / vnpy 经典的 Python 回测/交易框架，灵活但纯代码、学习曲线陡。Vibe-Trading 不跟它们拼底层引擎，而是用"自然语言 + AI 自动写策略"把这层门槛抹掉（还能把策略导出成 vnpy 格式）。
TradingView 看图、画线、写 Pine Script 策略的主流平台，可视化强但要自己写脚本。Vibe-Trading 能一键把策略导出成 Pine Script/通达信/MT5，更像是它的"上游策略生成器"。
HKUDS/AI-Trader（同门） 同一个港大实验室的另一个项目，主打"100% 全自动 agent 原生交易"。和 Vibe-Trading 是互补关系：Vibe-Trading 偏"研究+受限实盘"，AI-Trader 偏"全自动交易/跟单"。
付费量化平台（聚宽/优矿/米筐等） 国内成熟的在线量化平台，数据全、有社区，但要订阅付费、且策略困在平台里。Vibe-Trading 是免费开源、本地自建，数据自己接、策略自己留。
📌
这个项目适合：

有交易想法但不会编程、想低成本验证的个人；想入门量化的学习者/教学场景；以及想让 AI 帮自己跑投研流程、复盘真实交易的散户。一句 pip install 就能起步，是目前"自然语言量化"里最完整的开源方案。


不适合：

想要"开箱即赚"的人——它是研究工具，不保证收益；追求超低延迟、高频实盘撮合的专业交易团队（它的实盘能力还是实验性的、偏研究）；以及完全不想碰命令行/部署的纯小白（虽然用着简单，但"装起来"仍需要一点动手或让 AI 帮忙）。
