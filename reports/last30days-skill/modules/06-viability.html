🛠️ 产品化可行性

一个不会写代码的 PM，能不能用这个项目做出东西来？

低门槛
上手难度

这就是一个 Claude Code 的 skill。在 Claude Code 里输入一行命令就能装好，然后直接对话使用。零配置开始，Reddit、Hacker News、Polymarket、GitHub 开箱即用。

低门槛
依赖链

基础功能完全免费。想解锁 X、YouTube、TikTok 等更多来源，只需要注册几个免费 API key 或者登录浏览器。不需要付费订阅就能用。

高适配
Vibe Coding 适配度

这是一个 skill，你通过自然语言跟它交互——"/last30days 张三"、"帮我对比一下 A 和 B"。你不需要写一行代码。它是面向对话使用的。

你怎么上手？三步

1
装 Claude Code

如果你还没装，按官网指引装好 Claude Code 终端工具。

2
装这个 skill

在 Claude Code 里输入 /plugin marketplace add mvanhorn/last30days-skill，然后 /plugin install last30days。搞定。

3
开始搜

输入 /last30days 你感兴趣的任何话题，第一次运行会自动弹出 setup wizard 帮你解锁更多来源。

💡
对你来说，最大的价值不是"装好它"

作为一个 PM 在探索期，你最大的收获不是把这个工具跑起来——而是理解它的产品设计思路：怎么把十几个互不相通的数据源变成一个用户体验流畅的产品？怎么用 AI agent 做"预研"而非被动搜索？怎么让搜索结果变得"可分享"而不只是一个聊天记录？这些产品决策，比代码本身更有价值。
