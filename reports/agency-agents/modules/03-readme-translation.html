The Agency: AI Specialists Ready to Transform Your Workflow

A complete AI agency at your fingertips — From frontend wizards to Reddit community ninjas, from whimsy injectors to reality checkers. Each agent is a specialized expert with personality, processes, and proven deliverables.

中文

The Agency：触手可及的 AI 专家团队，随时改造你的工作流

一个完整的 AI 服务公司就在你指尖——从前端魔法师到 Reddit 社区运营忍者，从趣味注入师到质量检验官。每个 agent 都是拥有个性、流程和经验证交付物的专业专家。

原文 What Is This?

Born from a Reddit thread and months of iteration. Each agent is: Specialized — deep expertise, not generic prompt templates; Personality-Driven — unique voice and approach; Deliverable-Focused — real code, processes, measurable outcomes; Production-Ready — battle-tested workflows.

Think of it as: Assembling your dream team, except they're AI specialists who never sleep, never complain, and always deliver.

中文 这是什么？

起源于一个 Reddit 帖子，经过数月迭代。每个 agent：专精（深度领域知识，不是通用 prompt 模板）；有性格（独特的沟通风格和方法论）；重交付（真实代码、流程、可度量成果）；可投产（经过实战验证的工作流）。

可以这么理解：组建你的梦之队，只不过他们是永不睡觉、从不抱怨、永远交活的 AI 专家。

原文 Quick Start — Use with Claude Code (Recommended)

./scripts/install.sh --tool claude-code — install all agents to your Claude Code directory. Or manually copy a category if you only want one division: cp engineering/*.md ~/.claude/agents/

中文 快速开始 — Claude Code 使用方式（推荐）

执行 ./scripts/install.sh --tool claude-code 将所有 agent 安装到 Claude Code 目录。如果只需要某个部门，手动复制：cp engineering/*.md ~/.claude/agents/

原文 Use with Other Tools

Supports GitHub Copilot, Antigravity, Gemini CLI, OpenCode, OpenClaw, Cursor, Aider, Windsurf, Kimi Code, Codex. Run ./scripts/convert.sh first, then ./scripts/install.sh --tool <tool-name>.

Install only the teams you need: ./scripts/install.sh --tool claude-code --division engineering,security

中文 其他工具支持

支持 18 种工具：GitHub Copilot、Antigravity、Gemini CLI、OpenCode、OpenClaw、Cursor、Aider、Windsurf、Kimi Code、Codex 等。先运行 ./scripts/convert.sh 生成适配文件，再指定工具安装。

可以只安装需要的团队：./scripts/install.sh --tool claude-code --division engineering,security

原文 Agent Design Philosophy

Strong Personality — each agent communicates with a distinct voice and methodology; Clear Deliverables — agents output tangible artifacts (code, docs, strategies, reports); Proven Workflows — agents follow established best practices for their domains.

中文 Agent 设计哲学

强人格：每个 agent 以独特的声音和方法论沟通；清晰交付物：agent 产出的是具体成果（代码、文档、策略、报告）；经验证的工作流：agent 遵循各自领域已验证的最佳实践。
