同类项目速览

「给 AI 配专家角色」这条赛道并不孤单，它在生态里处于什么位置

contains-studio/agents 这套 The Agency 的角色风格（前端魔法师、奇趣注入师、Reddit 社区运营等命名）与早期爆火的 contains-studio agents 一脉相承，可以理解为同一思路的「加强扩张版」——后者更精简，前者把规模堆到了 232 个。
wshobson/agents 另一个高人气的 Claude Code subagents 合集，风格更偏「工程严谨」、少人格戏份。适合纯技术团队；The Agency 则覆盖面更广（含营销、销售、财务、游戏）、更重人设。
VoltAgent / awesome-claude-code-subagents 同类的「subagent 大合集」，主打数量与分类齐全。和它们比，The Agency 的差异化在于「原生 App 一键多工具安装 + 自动更新」这套分发体验，对非技术用户更友好。
各类「Awesome Prompts」清单 最传统的对手——一长串 Markdown 提示词清单。差距在于：清单只给你「一句话」，The Agency 给的是「人设+流程+交付标准+安装方式」的完整角色，可用性高一个量级。
📌
这个项目适合：

已经在用 Claude Code / Cursor 这类 AI 工具、想要「按岗位点名专家」的个人和小团队；想找现成角色模板来抄思路的产品/运营；喜欢开源、想自己改写成中文业务版的人。


不适合：

完全不用 AI 编程工具、只想要个聊天机器人的人；需要中文私域/微信原生场景开箱即用的团队（得自己改造）；追求「少而精、每个都打磨过」的洁癖型用户（这里是大而全，质量参差）。
