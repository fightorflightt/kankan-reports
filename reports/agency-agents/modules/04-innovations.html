04

创新点

和常规的「Prompt 模板库」比，它有什么本质不同？

这不是 prompt 合集，这是一套「AI 人才操作系统」

GitHub 上有无数个「Awesome ChatGPT Prompts」类的仓库。The Agency 做出了本质差异：它把 prompt 升级成了 agent——不是一段话，而是一个有身份、有流程、有交付标准的可安装文件。

🔄
工具无关的跨平台转换引擎

这是最核心的创新。大多数 agent 库只绑定一个工具，而它通过 convert.sh 脚本将同一套 agent 定义映射到 18 种工具的本地格式。这相当于「一次编写，到处运行」——你的 AI 团队不随工具迁移而丢失。

🏢
「完整公司」而不是「单一助手」

232 个角色覆盖了从写代码（工程部）到做设计（设计部）到搞营销（营销部）到管安全（安全部）到做游戏（游戏开发部）的全链条。别的 prompt 库给你一个「万能助手」，它给你一整家公司——你可以按部门选人、按项目组队。

📋
像招聘一样的 agent 设计

每个 agent 文件都有：身份与性格、核心使命、工作流程、交付物标准、沟通风格。这不像写 prompt，更像在写一份职位描述（JD）。用户可以像「面试」一样选择一个 agent：看它的专长、看它过往的交付标准、看它适不适合当前项目。

🇨🇳
深度的中国市场覆盖

极其少见的一点：232 个角色中包含了微信小程序开发者、小红书运营、飞书集成开发、知乎策略师、百度 SEO、抖音/B站/快手策略师等大量中国市场专属角色。这不是一个「翻译」出来的仓库，而是有真实中国市场场景思考的全球化产品。中文 fork 版（agency-agents-zh）更是增加了 50 个中国市场原创 agent。
